# 前端页面速查手册

路由定义在 `frontend/src/App.tsx`，共 6 个页面。

---

## 1. LobbyPage — 大厅工作台

**路由：** `/`  
**文件：** [LobbyPage.tsx](src/pages/LobbyPage.tsx)

用户打开 App 看到的第一个页面，相当于"主页/仪表盘"。

**布局：**
- **左侧栏**：品牌名、用户头像（Alex Chen）、导航菜单（MY COURSES / SETTINGS）、搜索框、新建录音按钮
- **主区域顶部**：标题 + 网格/列表视图切换按钮
- **主区域中部**：课程卡片列表，支持两种视图
  - **网格视图**：每张卡片展示课程缩略色块、课程名、讲次名、时长、笔记数
  - **列表视图**：表格形式，额外展示文件夹标签、日期
  - 有一张"处理中"状态的卡片（灰色骨架屏 + 动画脉冲）
- **主区域底部**：三个洞察卡片（Study Velocity / AI Insights / LIVE AI Courses）

**核心交互：**
- 点击"Upload the record"按钮 → 弹出 NewClassModal（上传 PPT + 音频的弹窗）
- 上传成功后跳转到 `/processing?session_id=...`
- 点击已完成的卡片 → 跳转到 `/notes/:id`

## 2. UploadPage — 文件上传页

**路由：** `/upload`  
**文件：** [UploadPage.tsx](src/pages/UploadPage.tsx)

功能上和 LobbyPage 里弹出的 NewClassModal 几乎一样，是一个全屏半透明遮罩 + 白色圆角卡片的弹窗式独立页面。

**布局：**
- 全屏毛玻璃背景遮罩
- 中央白色卡片，包含：
  - 标题：ACTION CENTER / New Class
  - 关闭按钮（返回上一页）
  - 两个上传区域（左：PPT/PDF，右：音频文件）
  - 上传中状态：进度条 + 步骤清单动画
  - 底部按钮：CANCEL / Save Workspace

**核心交互：**
- 拖拽或点击上传文件
- 上传验证（格式检查、大小限制 500MB）
- 提交后跳转到 `/processing?session_id=...`

## 3. SessionPage — 课中实时录音页

**路由：** `/session`  
**文件：** [SessionPage.tsx](src/pages/SessionPage.tsx)

学生上课时使用的核心页面，边录音边看 PPT 边记笔记。

**布局（三栏）：**
- **顶部导航栏**：LiberStudy logo + 导航 + 通知铃铛 + 用户头像
- **左侧栏（200px）**：幻灯片缩略图列表（LECTURE SLIDES），点击跳转到对应页
- **中间主区域**：PPT 画布
  - 顶部工具栏：翻页按钮、页码、缩放按钮、更换 PPT 按钮
  - 画布区域：渲染幻灯片图片，支持点击添加 inline 文字批注
- **右侧面板（320px）**：
  - 顶部：录音状态（REC + 计时器）+ 录音控制按钮
  - Pill 切换：My Notes / AI Notes
  - 笔记内容区：显示当前页的批注列表，或"录音结束后生成 AI 笔记"占位
  - 底部：文字输入框 + 结束录音·生成笔记按钮

**特殊弹窗：**
- **恢复弹窗**：检测到上次未完成录音时弹出，选项：继续录音 / 用现有录音生成笔记 / 放弃录音

## 4. ProcessingPage — 处理进度等待页

**路由：** `/processing?session_id=...`  
**文件：** [ProcessingPage.tsx](src/pages/ProcessingPage.tsx)

文件上传后的等待页面，展示后台处理进度，全部完成后自动跳转。

**布局：**
- 居中白色卡片
- 标题：正在处理课堂录音 + 预计剩余时间
- 进度条（蓝色）
- 四个步骤的有序列表，每步有三种状态：
  - ✓ 绿色圆圈：已完成
  - 蓝色脉冲圆圈：进行中（显示描述文字）
  - 灰色圆圈：待处理

**四个步骤：**
1. 语音转录（ASR）
2. PPT 解析
3. 时间轴构建（语义对齐）
4. 笔记生成

**失败状态：**
- 显示错误提示 + "重新处理"按钮

**轮询逻辑：**
- 每 2 秒请求 `/api/sessions/:id`，状态变为 `ready` 或 `partial_ready` 时自动跳转到 `/notes/:id`

## 5. NotesPage — 笔记主视图

**路由：** `/notes/:id`  
**文件：** [NotesPage.tsx](src/pages/NotesPage.tsx)

处理完成后的核心阅读页面，查看 AI 生成的笔记，三栏布局。

**布局（三栏）：**
- **顶部导航栏**：带通知铃铛和头像
- **左侧栏（200px）**：幻灯片缩略图导航（LECTURE SLIDES）
  - 点击缩略图可滚动中间画布到对应页
  - 当前页高亮（白色 2px 描边）
- **中间主区域**：PPT/PDF 画布（垂直滚动，所有页叠放）
  - 工具栏：翻页、缩放、导出 Markdown 按钮
  - 每页卡片上有：播放时间戳按钮、页码标签、低置信度警告标签
- **右侧面板（320px）**：
  - Pill 切换：My Notes / AI Notes
  - My Notes：展示该页用户批注（带时间戳）
  - AI Notes：展示用户批注 + AI 扩写说明（ACTIVE ANNOTATION）+ AI 要点（PASSIVE NOTES bullets，可点击跳转音频）
  - 底部：文字输入框 + 发送按钮

**其他功能：**
- 点击 bullet 时间戳可跳转音频播放位置
- 复制当前页笔记到剪贴板（显示 Toast 提示）
- 导出全部笔记为 Markdown 文件（下载到本地）

## 6. DetailedNotePage — 单页笔记详情

**路由：** `/notes/detail/:id`  
**文件：** [DetailedNotePage.tsx](src/pages/DetailedNotePage.tsx)

以"文章阅读"形式展示某一页笔记的完整内容，像读一篇排版精美的文章。

**布局（两栏）：**
- **顶部导航栏**：当前页高亮"Detailed Note"
- **左侧栏（200px，吸附固定）**：目录导航
  - 返回按钮（返回上一页）
  - 各章节标题（点击平滑滚动定位）
  - AI 说明入口
- **右侧主内容区**：
  - 顶部元信息：学科标签、日期、Session ID
  - 大标题（H1）
  - 正文内容，支持多种块类型：
    - `heading`：章节小标题（带下划线分隔线）
    - `paragraph`：正文段落
    - `bullets`：要点列表（圆点）
    - `definition`：术语定义卡片（灰色背景 + 代码样式术语名）
  - AI 补充说明区块（带左侧彩色边框）

**注意：** 当前数据为硬编码 Mock，尚未接真实 API。

## 页面跳转关系

```
LobbyPage (/)
  ├─ 点击课程卡片 → NotesPage (/notes/:id)
  └─ 点击"Upload the record" → [弹窗] → ProcessingPage (/processing)
                                            └─ 完成 → NotesPage (/notes/:id)
                                                         └─ [暂未接通] DetailedNotePage (/notes/detail/:id)

UploadPage (/upload)
  └─ 提交 → ProcessingPage (/processing)

SessionPage (/session)
  └─ 结束录音 → ProcessingPage (/processing)
```
