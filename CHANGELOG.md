## v0.4 — 2026-04-09

**背景**：Q1-Q10 需求澄清后，对 PRD 进行系统性修正，清除设计残留并对齐最新产品决策。

### 交互设计修正
- **删除连线/贝塞尔曲线设计**：图钉+连线方案已废弃，全文清除（11.2、11.3、11.4节）
- **统一批注概念**：将所有"图钉（Pin）"替换为"就地文本批注"（类 Edge PDF 批注），共 10+ 处
- **视图切换器位置确认**：统一为"右栏顶部药丸型分段控件"，删除错误的"右上角 Toggle"描述
- **PPT 浏览方式改为垂直滚动**：所有页面垂直排列如浏览网页，"当前页"定义为视口中占比最大的页面

### 产品逻辑修正
- **PPT 改为可选项**：无 PPT → 仅主动学习（录音+用户笔记→AI扩写）；有 PPT → 额外激活被动学习
- **AI 笔记模板②③重新定义**：
  - ② 全 PPT 讲解笔记：无需用户笔记，AI 纯基于录音稿 + PPT 自动生成
  - ③ 完整综合笔记：用户笔记 + PPT + 录音稿三者合并

### 页面关闭与恢复逻辑（新增）
- **关闭页面行为**：录音立即停止，后端处理不继续；音频切片持久化到 IndexedDB
- **新增 Story 1A-2**：会话中断恢复弹窗，下次打开时提供三个选项：继续录音 / 用现有录音生成笔记 / 放弃

### 部署方式变更
- 从"本地运行（localhost）/面试演示"改为**云端前后端分离部署**
- 删除所有 `make dev`、"面试官克隆仓库"等本地演示相关描述
- API Key 改为服务器端环境变量管理
- 成本表格更新：带宽/存储均按云端口径描述
- CLAUDE.md 同步更新上述决策
