# LiberStudy 配色规范 (Color Palette)

> 风格定位：温和、复古、纸质感（Paper-like）极简风格

---

## 核心色彩规范

| 色值 (HEX) | 颜色名称 | 角色定义 | 应用场景 |
| :--- | :--- | :--- | :--- |
| `#FFFFFF` | 纯白 (Pure White) | **主卡片/内容背景** | 最高层级容器背景，承载核心文本，确保最高阅读对比度 |
| `#F7F7F2` | 极浅米白 (Off-White) | **次级背景 Bg-Level 1** | 与纯白卡片产生微妙区分的背景区域，主要内容区底衬，减少视觉疲劳 |
| `#F2F2EC` | 浅米色 (Light Beige) | **辅助背景 Bg-Level 2** | 次级组件/模块背景、快捷键提示框底色、轻度 Hover 反馈 |
| `#EAE9E0` | 米灰色 (Beige Base) | **全局底色 App Base** | 全局最底层背景（页面留白区域、侧边栏），奠定温暖复古基调 |
| `#E3E3DA` | 中米灰 (Medium Beige) | **浅色分割线/描边** | 较弱的区域划分、组件边缘细线描边、Disabled 状态底色 |
| `#D0CFC5` | 暖灰色 (Warm Gray) | **深色描边/次要元素** | 明显边框轮廓、图标线条、非激活状态占位符文本颜色 |
| `#798C00` | 橄榄绿 (Olive Accent) | **强调色/品牌色** | 核心视觉焦点：引导视线、关键信息强调、Active 状态高亮标识 |
| `#292929` | 深炭灰 (Charcoal) | **主文本色** | 带微弱暖色调的深灰，比纯黑更柔和，契合纸质感主题 |
| `#72726E` | 中性灰 (Neutral Gray) | **次要文本色** | 日期、标签、辅助说明等次级信息 |

---

## 背景层级系统

> 从底层到顶层按照颜色明度递进，模拟物理纸张堆叠的空间感

```
底层 (Base)       #EAE9E0  ████
中层 (Surface)    #F7F7F2  ████  或  #F2F2EC  ████
顶层卡片 (Card)   #FFFFFF  ████
```

---

## 边框与分割原则

- 尽量少用粗重的深色边框
- 使用 `#E3E3DA` 或 `#D0CFC5` 作为 1px 细线分割
- 保持界面的呼吸感和通透性

---

## 强调色使用原则

- `#798C00` 橄榄绿为唯一强调色，克制使用
- 适用场景：Active 状态指示线、关键操作按钮、进度/高亮标记
- 避免大面积使用，保持其稀缺性和视觉引导作用
