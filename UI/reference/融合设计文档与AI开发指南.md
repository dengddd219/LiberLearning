# **LiberStudy 核心功能架构、原型与 UI 设计指南 (融合完整版)**

版本: v0.3 | 日期: 2026-04-08

本文档结合了核心功能决策与 UI 设计规范，作为 Figma 低保真原型绘制以及前端 AI 代理开发的直接依据。整个系统采用 Web 响应式设计（优先桌面端），核心围绕“全局侧边栏与大厅”、“课中采集”和“课后笔记查看”三大模块展开。

## **0\. 全局设计原则与视觉规范 (UI 基座)**

* **MVP UI 目标**：最简可验证全流程，不追求视觉精美，核心是功能闭环跑通。整体风格现代、极简、专业；"纸质"内容优先的无干扰感；扁平化图标，薄描边，克制的色彩使用。  
* **组件库与图标**：图标库统一使用 lucide-react，统一薄描边风格（strokeWidth={1.5}），图标尺寸常规为 w-5 h-5，紧凑区域 w-4 h-4。图标主色 text-gray-700，禁用态 text-gray-400。  
* **颜色规范 (Tailwind)**：  
  * 页面/工具栏背景：bg-white / bg-gray-50 (\#F9FAFB)  
  * 边框/分隔线：border-gray-200 / bg-gray-300  
  * 主文字：text-gray-900；次要文字/提示：text-gray-500；Hover 背景：hover:bg-gray-200  
* **警告标识 ⚠️**：系统警告黄，用于对齐置信度低的页面。  
* **字体**：正文字体使用系统无衬线体（Tailwind 默认），text-sm。

**1\. 全局侧边栏与大厅工作台（Lobby Workspace \- 首页）**

废弃了原有的纯平铺网格，引入左侧轻量级单层侧边栏与右侧内容区的双栏结构，提升中后期的课程管理效率。

* **全局轻量级侧边栏 (Lightweight Sidebar)**：  
  * **顶部常驻核心动线**：包含全局快捷搜索 (Ctrl+K)，主页/大厅入口（汇总展示所有近期课程），以及固定且高亮的 \+ 新建课堂 / 🎙️ 开始录音 主动作按钮。  
  * **我的课程区 (My Courses)**：采用单层文件夹设计（杜绝无限嵌套）。提供 \+ 新建课程 按钮，用户可创建诸如“MSBA 7028”或“商业分析”的单层课程夹。点击具体课程夹，右侧主工作区自动过滤对应内容。  
  * **底部常驻区**：设置入口与用户个人头像、额度信息。  
* **主工作区顶部导航**：右侧设有一个分段控制器 (Segmented Control)，用于切换“网格 (Grid)”与“列表 (List)”视图。  
* **主工作区展示双轨制 (视图切换)**：  
  * **网格视图 (Grid View)**：保留视觉记忆优先的心智。封面为 PPT 第一页的预览图。卡片信息包含课程标题（可双击重命名）、录音时长、笔记数量、最后访问时间。  
  * **列表视图 (List View)**：注重效率与高信息密度（类似 Notion 数据库行）。按表头列展示：极小缩略图 \+ 课程名称、所属课程夹 (Folder)、上课日期、时长与笔记状态。  
* **卡片状态表现**：解析中（骨架屏扫光+进度条）、已完成（正常展示）、异常（标红警告+重试/删除按钮）。Hover 时浮现菜单（重命名、导出、删除）。  
* **空状态界面**：无数据时居中展示插画与“上传第一节课”主按钮，以及可选的“体验示例课程”入口。

**2\. 课中采集界面（上课场景 / 核心工作区）**

这是 MVP-0 的核心场景，分为“有 PPT”和“无 PPT”两种视图。

### **模式 A：有 PPT 模式（全局三栏布局）**

* **左栏（宽度约 200px）**：可折叠大纲导航。默认收起显示图标，Hover 时向右浮出完整宽度的 PPT 缩略图列表，点击可平滑滚动定位。  
* **中栏（弹性自适应）**：PPT 交互主画布。垂直滚动浏览所有页面。  
  * **PDF 阅读器风格顶部工具栏**：背景色设为 bg-gray-50 (\#F9FAFB)，下边框 border-b border-gray-200，组内间距 gap-1 或 gap-2，组间用 \<div className="w-px h-5 bg-gray-300 mx-2" /\> 分隔。从左至右包含：  
    * ① 导航：PanelLeft（侧栏开关）。  
    * ② 批注工具：Highlighter（高亮）+ 下拉、Pen（绘制）+ 下拉、Eraser（橡皮擦）、Type（文本框就地批注）。  
    * ③ 辅助功能：Volume2（朗读）、Languages（翻译）。  
    * ④ 缩放：Minus、Plus、Maximize2。  
    * ⑤ 页码（居中区）：输入框（宽 w-10）与总页数。  
    * ⑥ 页面视图：RotateCw、BookOpen。  
    * ⑦ 右对齐操作（ml-auto）：Search、Printer、Save、SaveAll、Maximize、Settings。  
  * **就地文本批注交互（核心）**：参考 Edge 浏览器。用户点击 PPT 任意位置，就地弹出输入框（位置对齐点击坐标）。输入确认后以“轻量标注样式”留在页面上。系统自动绑定页码、当前录音相对时间戳和 Y 轴坐标。当前页批注会即时同步出现在右栏笔记区。  
* **右栏（固定约 320px）**：动态笔记区。  
  * **录音控制条**：顶部固定，包含开始/暂停/结束按钮及录音时长计时器。  
  * **笔记区**：实时同步展示中栏的就地批注内容，用户也可直接在此区域空白处键入笔记（与画布双向同步）。视口滑动翻页时，右栏自动切换为新页面的专属笔记。

### **模式 B：无 PPT 纯笔记模式**

* **两栏布局**：隐藏中栏画布，仅保留左侧的录音控制区和右侧的自由富文本笔记输入区。不与页码绑定，按时间顺序和话题段落记录。

**3\. 课中全局悬浮功能与异常处理**

* **Quick Ask（课中快问）**：悬浮球采用 AssistiveTouch 风格，常驻右下角，可拖拽并吸附边缘。点击展开对话框后，顶部带标题和关闭按钮，中间为对话历史流（左侧 AI 白底，右侧用户灰底），底部为输入框（输入 ? 可自动触发近 N 分钟摘要）。  
* **会话中断恢复弹窗**：意外关闭后重新进入页面触发，显示未完成的录音时长和笔记数，提供三个操作项：“继续录音”、“用现有录音生成笔记”、“放弃这次录音”。

## ---

**4\. 课后文件上传与处理进度（针对场景①或网络重连）**

* **并排上传区**：左右双上传区（音频文件限制 ≤2GB，PPT 文件限制 ≤100MB），支持拖拽，两个文件均上传成功后“开始处理”按钮激活。  
* **处理进度页**：线性进度条、当前阶段状态文案（语音转录中 / PPT 内容解析中 / 翻页时间轴构建中 / 笔记生成中）、预估剩余时间。  
  * **流式生成交互**：进入「笔记生成中」阶段后，进度文字切换为逐页计数格式（例：第 3/18 页笔记生成中...）。每页生成完成后立即在笔记查看界面渲染该页笔记，支持流式逐页展示，用户无需等待全部完成即可提前查看。  
* **手动打点辅助（可选）**：如果系统推断失败，提供一个可选界面让用户标记音频时间点对应哪一页 PPT。

**5\. 课后笔记查看界面（消费场景）**

复用上课的三栏布局（左 200px / 中自适应 / 右 320px），但右侧笔记区功能极大增强。

### **右栏顶部操作区：药丸型笔记切换控件**

采用紧凑的无边框药丸形容器，背景为暖灰色/米色（近似 \#FAF9F7），营造“纸质”无干扰感。

* **“我的笔记”视图（左分段）**：极致极简，只显示用户上课时真实记录的骨架关键词。采用 List 图标，激活态背景与容器同色，Hover 时浮出提示 **"My note"**。  
* **“AI 笔记”视图（右分段/默认态）**：展示被动学习（PPT 骨干）+ 主动学习（根据用户笔记扩写）的叠加结构化笔记。  
  * **样式**：激活态为纯白色，带微妙阴影。采用 Sparkles 图标 \+ 文字 \+ ChevronDown。  
  * **严格字色规范**：笔记内容不使用背景色块。**用户原始笔记**严格使用纯黑（text-gray-950 / text-black）；**AI 生成内容**严格使用深灰（text-gray-700），禁止使用更浅的颜色以保证可读性。

### **AI 笔记控制面板**

点击右侧雪佛龙触发下拉菜单（半透明干净卡片），切换当前页重渲染模型：

* **下拉模板选择（4种）**：  
  1. **基于我的笔记扩写**：用户批注关键词/疑问 \+ 转录稿 → 扩写结构化段落（无批注页面此项灰显）。  
  2. **全 PPT 讲解笔记**：PPT bullet \+ 转录稿 → 行级 LLM 注释 \+ page\_supplement。  
  3. **完整综合笔记**：按页生成最完整版本，结合前两者。  
  4. **大纲摘要**：跨所有页面提取核心概念，生成一页式全课概览。  
* **粒度切换**：“简单”（1-2句概括）与“详细”（保留例证长段落）。

### **内容卡片互动**

每条笔记附带引用时间戳（如 \[00:03:21\]），点击可拉起音频播放器并跳转播放。包含 PPT Bullet 行级注释、用户原始笔记、扩写内容以及老师额外补充。生成失败的页面显示单页重试按钮。

### **导出与复制模块**

* **一键复制单页**：右栏顶部工具行放置复制按钮（Copy 图标，w-4 h-4）。点击复制当前页 Markdown，并显示 1.5 秒「已复制」Toast。  
* **全局导出规范**：内容为**纯文字**，不嵌截图。使用 jsPDF+html2canvas 导出大纲式排版 PDF，或导出带有严格格式的 Markdown。
